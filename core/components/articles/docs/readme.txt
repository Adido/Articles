First Released: November 28th, 2011
Author: Shaun McCormick <shaun+articles@modx.com>
License: GNU GPLv2 (or later at your option)

This component is a Custom Resource Class for MODX 2.2+. It allows you to easily create blogs on your sites complete
with a custom UI for managing them and blog posts. It automatically handles URL archiving, RSS feeds, comments, tags
and more.

Please read the official documentation at:
http://rtfm.modx.com/display/ADDON/Articles
